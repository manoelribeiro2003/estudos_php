Aqui está a parte de Grid System. O grid system do bootstrap funciona na divisão de grades de 12 unidades.
O sistema de grade é responsivo usando algumas classes como: col-, col-sm|md|lg|xl|xxl-. Tudo isso para se adequar ao tamanho das telas.
sm = small devices
md = medium devices
lg = large devices
xl = xlarge devices
xxl = xxlarge devices

De acordo com o tamanho da tela, o conteudo vai ficando responsivo.
